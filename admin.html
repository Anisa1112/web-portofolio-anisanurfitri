⬇ DOWNLOAD data.js

Klik tombol di bawah untuk men-download file data.js yang berisi semua data project dan sertifikat terkini. File ini akan menggantikan data.js lama di folder portfolio kamu.

Langkah selanjutnya setelah download

1
Buka folder repo GitHub kamu di komputer. Cari file data.js lama.
2
Replace dengan file data.js yang baru saja kamu download dari sini.
3
Jalankan perintah: git add data.js → git commit -m "update data" → git push
4
Tunggu ±1 menit, lalu buka GitHub Pages kamu — portfolio langsung update! 🎉
